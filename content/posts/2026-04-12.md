This is the first post written in Markdown. The pipeline reads `.md` files from `content/posts/`, parses the front matter, converts the content to HTML, and renders it with the same layout as the rest of the site.

## What works

- Headings, paragraphs, and horizontal rules
- Lists like this one
- **Bold** and *italic* text
- `inline code` and code blocks
- [Links](https://example.com)
- Blockquotes

---

> Writing in Markdown is much faster than writing raw JSX.
